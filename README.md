# Pushing repositories
###Instructions

####Setting up new Repository 
```
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:marwai/help.git
git push -u origin master
```
#### Pushing commits

```
git add .
git commit -m "comment"
git push
```
